# 🏦 Анализ банковских транзакций

## 📋 Описание проекта
Анализ 1 миллиона транзакций клиентов банка для выявления закономерностей в финансовом поведении пользователей.

## 📊 Данные
- **Источник**: Google Drive
- **Объем**: 1 000 000 транзакций
- **Клиенты**: 2000 уникальных клиентов с известным полом

## 📁 Структура данных
| Поле | Описание |
|------|----------|
| customer_id | Идентификатор клиента |
| tr_type | Тип транзакции |
| amount | Сумма (положительная - доход, отрицательная - расход) |
| mcc_code | Код MCC (Merchant Category Code) |
| tr_description | Описание типа транзакции |

## 🎯 Выполненные задачи

### Задание 1: Анализ POS и ATM транзакций
Определена доля транзакций, совершенных через POS-терминалы или банкоматы.

### Задание 2: Топ-10 типов транзакций
Выявлены наиболее частые типы операций клиентов.

### Задание 3: Максимальные доходы и расходы
Найдены клиенты с максимальными доходами и расходами.

### Задание 4: Статистика по суммам
Рассчитаны средние и медианные значения сумм для разных категорий.

### Задание 5: Гендерный анализ
Сравнены средние траты и поступления мужчин и женщин.
